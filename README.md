A fully functional Rock Paper Scissors web game where users play against the computer. 
This project demonstrates core JavaScript concepts like event listeners, random number generation, and real-time UI updates.

Key Features:
Interactive Gameplay: Play against an automated computer opponent.
Real-time Score Tracking: Scoreboard updates instantly for both the user and the computer.
Dynamic Messaging: Displays game status (Win, Loss, or Draw) with color-coded feedback.
Modular Code: Built using clean, logic-driven JavaScript functions for win-condition checking.
